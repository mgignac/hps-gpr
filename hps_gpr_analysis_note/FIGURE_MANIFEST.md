## Notes

- The LaTeX note uses the final Overleaf destinations above, not the repository-root
  source paths.
- Missing files are rendered as placeholder boxes by the macro
  `\graphicorplaceholder{...}{...}` so the note compiles even before the PNGs are
  copied.
- For the final Overleaf project, copy each available local figure into the listed
  destination path relative to the project root.
