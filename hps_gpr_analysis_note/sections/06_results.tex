\section{Results}
\label{sec:results}

This section separates two result layers. The standalone 2015, 2016 10\%, and
2015$+$2016 subsections are retained as the current validated baseline summaries for
the individual and early-combination stages of the workflow. The review-facing update,
however, is the shared-$\eps^2$ combination of 2015, 2016 10\%, and 2021 1\%, for
which the note now uses the current `summary\_combined\_all\_rad\_penalty` summary. In
that three-dataset study the radiative-fraction penalty is applied consistently to all
active datasets, and the combined global-significance curve is the only global
look-elsewhere representation interpreted in the note.

\subsection{2015 analysis}
\label{sec:results-2015}

The 2015 analysis uses the fully unblinded engineering-run spectrum over
\SIrange{20}{130}{MeV}. Relative to the published local-window search, the present GPR
study is being validated over a substantially wider mass interval while preserving a
smooth, well-behaved upper-limit curve. This is an important methodological point: the
value of the GP approach is not merely that it fits the local background, but that it
does so in a way that remains stable when the search window is widened and the local
fit boundaries of the earlier polynomial approach are removed.
Figure~\ref{fig:results-2015-ul} shows the corresponding observed and expected upper
limits, while Figure~\ref{fig:results-2015-significance} separates the toy-tail
diagnostic from the analytic $p_0$ significance summary.

\subsection{2015+2016 10\% combined upper limit on \texorpdfstring{$\eps^2$}{epsilon2}}
\label{sec:results-combined-1516}

2016 10\% development sample. This is already a meaningful milestone for the analysis
because it shows that the GPR framework can combine statistically independent data sets
with different resolutions, search ranges, and radiative normalizations inside a single
likelihood rather than by comparing precomputed limit curves after the fact.

single data set. For that reason the combined result is quoted directly as an

overlay has been frozen. Figure~\ref{fig:results-combined-1516-ul} shows the first

\subsection{Combined limit on \texorpdfstring{$\eps^2$}{epsilon2} using 2015+2016 10\%+2021 1\%}
\label{sec:results-combined-151621-1pct}

This three-sample study combines the full 2015 data set, the 2016 10\% development
subset, and the 2021 1\% subset using the leakage-retaining signal template and the
radiative-fraction penalty described in Eq.~\eqref{eq:frad-penalty}. In the current
configuration the baseline radiative fractions are $f_{\mathrm{rad}}=0.085$ for 2015
and $f_{\mathrm{rad}}=0.05$ for both 2016 and 2021, with a common
$\delta_f=0.07$ penalty applied to each dataset through
$f_{\mathrm{rad}}^{\mathrm{eff}}=(1-\delta_f)f_{\mathrm{rad}}$. At fixed extracted
signal yield this weakens the corresponding $\eps^2$ constraint by a factor of
$1/(1-\delta_f)\simeq 1.075$, so the combined limit shown here should be interpreted as
the conservative systematic-penalized baseline rather than as the no-penalty central
value.

This subsection should therefore be read as the systematic-penalized normalization
scenario with $\delta_f=0.07$ applied coherently to each active dataset, not as the
no-penalty central-value presentation.

The observed combined $\eps^2$ limit in Figure~\ref{fig:results-combined-151621-1pct-ul}
is smooth and remains close to the expected band over most of the scan. That behavior
is consistent with a statistically stable shared-$\eps^2$ analysis. Changes in slope
should not be overinterpreted as narrow resonant structure, because the active dataset
content varies across the mass scan. Only the approximate \SIrange{35}{130}{MeV}
interval is a genuine three-way overlap. Below that range the result interpolates
between 2015-only and 2015$+$2021 support, while above it the analysis transitions to a
2016$+$2021 combination and finally to a 2021-dominated tail near the upper edge of the
scan.

The significance diagnostics in Figure~\ref{fig:results-combined-151621-1pct-significance}
must be read with the definitions of Section~\ref{sec:methodology} in mind. The toy-tail
quantities $p_{\rm strong}$, $p_{\rm weak}$, and $p_{\rm two}$ test how unusual the
\emph{observed limit} is relative to the background-only limit ensemble, whereas the
analytic $p_0$ curve is the local bump-hunt statistic obtained from the profiled shared
likelihood. These are intentionally different questions. The combined local/global
$p_0$ representation in the radiative-penalty suite is the correct place to discuss the
look-elsewhere effect for the shared search; by contrast, the stacked single-dataset
panels of Figure~\ref{fig:local-significances-across-datasets} are included only to
show local fluctuation structure and not to support dataset-level global-significance
claims.

The main feature of the present three-way summary remains the narrow local enhancement
in the combined analytic-significance curve near \SI{126}{MeV}. A combined local excess
larger than any individual input fluctuation is qualitatively possible because the
shared-$\eps^2$ fit pools statistically independent information with mass-dependent
weights, but the feature is still best treated as a review target rather than a
physics claim. The conservative reading is that the systematic-penalized combined
\emph{limit} behaves sensibly and that the combined global-significance conversion has
the correct form for this stage of the analysis, while the individual-dataset auxiliary
overlays and the final scan-level calibration still await a frozen rerun with the
updated plotting code. Figure~\ref{fig:results-combined-151621-1pct-display} provides a
representative observed-data extraction display for that same shared-coupling fit.

\caption{Observed and expected 95\% CL upper limit on $\eps^2$ for the

The projected unblinded reach in Figure~\ref{fig:results-combined-151621-1pct-projection}
is obtained from the same dataset-level $\eps^2$ curves using explicit
$1/\sqrt{L}$ rescaling before the shared-coupling combination is recomputed. In the
current projection, the fully unblinded 2015 sample is held fixed, the 2016 10\%
subset is promoted by a factor of 10 in statistics, and the 2021 1\% subset is
promoted by a factor of 100. At each mass, the constituent curves are therefore scaled
dataset by dataset and then recombined through the same inverse-variance logic used in
the live projection code. Writing $x_d(m)\equiv\eps^2_d(m)$ for the dataset-level
curve,
\begin{equation}
\frac{1}{x_{\mathrm{proj}}(m)^2}
=
\sum_d \frac{L_d/L_{d,0}}{x_d(m)^2},
\end{equation}
so the projected combined limit is
\begin{equation}
\eps^2_{\mathrm{proj}}(m) = \left[\sum_d
\frac{L_d/L_{d,0}}{\left(\eps^2_d(m)\right)^2}\right]^{-1/2}.
\end{equation}
The consequence is that the projection changes shape, not only normalization: the
highest-statistics 2021 input dominates much of the upper-mass reach, while extra gains
still appear where multiple constituent datasets simultaneously occupy relatively
low-background regions.
