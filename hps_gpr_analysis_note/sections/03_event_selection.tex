\section{Event Selection}
\label{sec:event-selection}

The resonance-search workflow in this package begins from histogrammed invariant-mass
distributions that already encode the relevant prompt-selection logic for each run
period. Because the current repository is focused on statistical inference rather than
detector reconstruction, the event-selection discussion is intentionally scaffolded but
should still be explicit about which ingredients are finalized and which remain under
active documentation.

The 2015 and 2016 selections shown in Table~\ref{tab:event-selection} are adapted from
the HPS internal prompt-resonance notes and from the prior rough draft maintained in
this repository. They are sufficiently mature to anchor the present statistical note.
By contrast, the 2021 column is kept deliberately provisional: the row structure is in
place, but several prompt-tight values still need to be synchronized with the final
2021 reconstruction note or publication.

The statistical interpretation in the remaining sections assumes that each histogram is
already defined after the corresponding prompt selection. No additional per-event cuts
are introduced by the GP package itself; the package only consumes the binned invariant
mass observable together with the dataset-specific scan range, mass resolution, and
radiative-fraction inputs.
