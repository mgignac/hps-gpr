\section{Introduction}
\label{sec:introduction}

The Heavy Photon Search experiment probes narrow vector resonances in electroproduced
$e^+e^-$ pairs. In this regime, the dominant analysis challenge is not the signal model
itself but the control of the smooth trident background and its correlated uncertainty
in the vicinity of each test mass. The present workflow addresses that problem with a
single global Gaussian-process-regression (GPR) model for each active dataset, combined
with a profiled multi-bin likelihood, expected \CLs\ bands, and a statistically
well-defined multi-dataset combination in a shared coupling parameter
$\eps^2$ \cite{HPS2015DarkPhoton,HPS2016PromptLong,HPSExperiment2022}.

This note is written from the current package implementation rather than from the older
piecewise drafts. In particular, the baseline local-significance calculation is the
profiled Gaussian multi-bin likelihood-ratio test, not the legacy blind-window
sum-count approximation. Likewise, the signal template is now treated as a
full-histogram Gaussian line shape whose blinded-bin slice enters the likelihood
without renormalization inside the blind region. That choice preserves the physically
useful interpretation that the signal parameter $A$ denotes the total local signal
yield of the narrow resonance rather than only the portion captured inside a finite
blind window.

\subsection{Relation to the published HPS prompt searches}

The published 2015 and 2016 HPS prompt-search results established the experimental
benchmark for narrow dark-photon searches in fixed-target $e^+e^-$ spectra. Those
results remain the public reference points for the prompt program and are reproduced in
Figure~\ref{fig:published-hps-context}. The analysis documented here should be viewed as
the next statistical step beyond that published baseline: the detector inputs remain
dataset-specific and conventional, but the background modeling, the treatment of
blinded-bin covariance, the expected-band workflow, and the shared-$\eps^2$
combination are all substantially more unified than in the original resonance-search
pipeline.

The move from earlier local-window and sideband-reconfiguration studies to the present
global-GP workflow was driven by validation experience inside this repository. First,
piecewise windows produced visibly less smooth expected and observed limit curves near
window boundaries. Second, local fit-range changes complicated the interpretation of
cross-dataset combinations because different datasets could enter or leave the local
model in slightly different ways at neighboring masses. Third, once the background is
represented as a coherent mean-plus-covariance object at every mass hypothesis, the
statistical combination becomes a true multi-channel likelihood problem rather than an
ad hoc averaging of per-dataset limits. In internal validation studies these choices
lead to markedly more stable scans and, in selected mass intervals, sensitivity gains
that can reach the order-of-magnitude level relative to earlier exploratory analysis
configurations. Because the leakage-retaining signal-template update documented in this
note still requires a final rerun of the main limit products, those gains are treated
here as provisional analysis-development observations rather than as final physics
claims.

\subsection{Scope of this note}

The first goal of the document is to provide a physics-analysis description of the
current package at a level suitable for collaboration review and later journal-facing
adaptation. The second goal is practical: the note is organized as a self-contained
Overleaf handoff with explicit figure destinations for upper-limit bands,
injection/extraction validation, representative fit displays, kernel studies, published
reference figures, and significance bookkeeping. Wherever the underlying 2021 inputs or
fully rerun products are not yet finalized, the corresponding sections are retained as
explicit placeholders rather than written as if the analysis were already complete.
