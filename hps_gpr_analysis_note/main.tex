\title{\vspace{-1.0em}HPS Gaussian-Process Resonance Search Analysis Note\\
\large Global background modeling, limit setting, toy validation, and shared-$\eps^2$ combination}
\author{Emrys Peets}
\date{March 7, 2026}

\begin{document}
\maketitle

\begin{abstract}
This note documents the current Heavy Photon Search (HPS) Gaussian-process-regression
(GPR) bump-hunt workflow as implemented in the modular \texttt{hps\_gpr} package and
validated in the \texttt{v15\_8} development notebook. The analysis performs global
background interpolation in the invariant-mass spectrum, local signal extraction with
Gaussian-profiled background nuisances, \CLs\ upper limits in signal-yield and
$\eps^2$ space, and statistically independent multi-dataset combinations in a shared
$\eps^2$ parameter of interest. The note is organized for Overleaf use and includes
explicit placeholders where public 2021 run metadata or final unblinded products are
not yet citable.
\end{abstract}
